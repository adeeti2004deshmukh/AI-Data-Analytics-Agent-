# AI-Powered Data Analytics Agent

An intelligent analytics platform that allows users to upload CSV datasets, ask questions in natural language, generate SQL queries automatically using Gemini AI, visualize results, and receive AI-generated business insights.

## Features

* Upload any CSV dataset
* Natural language to SQL conversion
* Dynamic SQLite query execution
* Interactive Plotly visualizations
* AI-generated business insights
* Download query results as CSV

## Tech Stack

* Python
* Streamlit
* Pandas
* SQLite
* Plotly
* Google Gemini API

## How to Run

Install dependencies:

pip install -r requirements.txt

Run the application:

streamlit run app.py
